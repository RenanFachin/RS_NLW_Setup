## O que é?
Front-end para a aplicação que registra os hábitos.

## Instalação
Faça o clone do repositório e acessar na pasta web

```bash
git clone https://github.com/RenanFachin/RS_NLW_Setup
cd server
```

Instalando as dependências
```bash
npm i
```

Rodando o servidor
```bash
npm run dev
```
